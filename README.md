# 일정 관리 앱 (Schedule App)

간단하게 일정을 추가하고 관리할 수 있는 웹 기반 일정 관리 앱입니다.  
GitHub Pages를 이용해 배포되었습니다.

---

## 🔗 배포 주소
https://awefiofaiw.github.io/schedule-app/

---

## ✨ 주요 기능
- 일정 추가 (제목 / 날짜 / 시간)
- 과거 날짜 일정 추가 제한
- 중복 일정 방지
- 오늘 일정과 전체 일정 분리 표시
- 일정 완료 체크
- 일정 삭제
- 필터 기능 (전체 / 미완료 / 완료)
- 일정 검색
- 월 단위 달력 표시
- 날짜 선택 시 해당 날짜 일정 표시
- localStorage를 이용한 데이터 저장 (새로고침 후 유지)
